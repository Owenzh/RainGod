flask实现的居民信息管理系统_雨神
=======
## 居民信息管理系统


#### 技术栈

- python3
- flask
- flask form
- flask sqlalchemy 
- sqlite数据库
- layui模板

#### 运行项目

```
#clone项目到本地
git clone https://github.com/Owenzh/RainGod.git
#安装依赖
pip install -r requirement.txt
#运行sql
python db.py
#启动项目
python manage.py runserver
#项目地址
http://127.0.0.1:8080
#默认超级管理员
root:root
#
```
